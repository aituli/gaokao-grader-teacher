# 数学阅卷示例

## 示例1：函数极值题

### 题目
已知函数 $f(x) = x^3 - 3x^2 + 2$，求函数的极值。

### 学生答案
$f'(x) = 3x^2 - 6x$

令 $f'(x) = 0$，得 $x = 0$ 或 $x = 2$

$f(0) = 2$，$f(2) = -2$

极大值是2，极小值是-2。

### 阅卷结果

**【评分结果】**
得分：9分（满分12分）

**【评分理由】**
- 求导正确：2分 ✓
- 令导数为零并求解：2分 ✓
- 计算函数值：2分 ✓
- 缺少判断极值类型的过程：扣3分
- 答题不够规范（缺少"解"字和必要说明）：扣1分

**【扣分说明】**
1. **极值判断过程缺失（扣3分）**
   - 学生只找到了临界点，但没有说明如何判断是极大值还是极小值
   - 应补充：通过二阶导数 $f''(x) = 6x - 6$，判断 $f''(0) = -6 < 0$（极大值），$f''(2) = 6 > 0$（极小值）
   - 或者通过一阶导数符号变化来判断

2. **答题规范性（扣1分）**
   - 解答题应写"解："
   - 关键步骤应配上文字说明

**【改进建议】**
1. 解答题开头写"解："
2. 求出临界点后，必须判断极值类型（二阶导数法或一阶导数符号法）
3. 关键步骤配上文字说明，如"令 $f'(x) = 0$，解得..."
4. 最后写明"综上所述，函数的极大值为...，极小值为..."

## 示例2：立体几何证明题

### 题目
（12分）如图，在四棱锥 $P-ABCD$ 中，底面 $ABCD$ 是正方形，$PA \perp$ 平面 $ABCD$，$PA = AB = 2$。

（1）求证：$BD \perp$ 平面 $PAC$；
（2）求点 $A$ 到平面 $PBD$ 的距离。

### 学生答案（第1问）
因为 $ABCD$ 是正方形，所以 $AC \perp BD$。

因为 $PA \perp$ 平面 $ABCD$，所以 $PA \perp BD$。

所以 $BD \perp$ 平面 $PAC$。

### 阅卷结果

**【评分结果】**
得分：5分（满分6分）

**【评分理由】**
- 证出 $AC \perp BD$：1分 ✓
- 证出 $PA \perp BD$：2分 ✓
- 得出 $BD \perp$ 平面 $PAC$：1分 ✓
- 缺少关键说明（线线垂直→线面垂直的判定条件）：扣1分

**【扣分说明】**
1. **缺少关键条件说明（扣1分）**
   - 学生直接得出 $BD \perp$ 平面 $PAC$，但没有说明 $AC$ 与 $PA$ 相交
   - 线面垂直判定定理要求：一条直线垂直于平面内两条相交直线
   - 应补充：因为 $AC \cap PA = A$，且 $AC, PA \subset$ 平面 $PAC$

**【改进建议】**
1. 立体几何证明要注意定理条件的完整性
2. 使用判定定理时，要明确指出相交条件
3. 建议格式：
   ```
   因为...（条件1）
   因为...（条件2）
   又...（相交条件）
   所以...（结论）
   ```
