# 物理阅卷示例

## 示例1：力学计算题

### 题目
（12分）如图所示，质量 $m = 2\text{kg}$ 的物体在水平拉力 $F = 10\text{N}$ 的作用下沿水平面做匀加速直线运动。已知物体与水平面间的动摩擦因数 $\mu = 0.3$，重力加速度 $g = 10\text{m/s}^2$。

（1）求物体运动的加速度大小；
（2）若物体从静止开始运动，求 $t = 3\text{s}$ 时的速度大小。

### 学生答案

（1）$F - f = ma$

$f = \mu mg = 0.3 \times 2 \times 10 = 6\text{N}$

$a = \frac{F - f}{m} = \frac{10 - 6}{2} = 2\text{m/s}^2$

（2）$v = at = 2 \times 3 = 6\text{m/s}$

**【改进建议】**

1. **规范答题格式**
   ```
   解：
   （1）对物体进行受力分析，物体受到拉力F、摩擦力f、重力mg和支持力N。
   在水平方向，根据牛顿第二定律：
   F - f = ma
   
   竖直方向：N = mg
   滑动摩擦力：f = μN = μmg = 0.3 × 2 × 10 N = 6 N
   
   代入得：a = (F - f)/m = (10 - 6)/2 m/s² = 2 m/s²
   
   （2）物体从静止开始做匀加速直线运动，初速度v₀ = 0
   根据运动学公式：v = v₀ + at
   代入数据：v = 0 + 2 × 3 m/s = 6 m/s
   ```

3. **注意单位**

   - 单位应使用国际单位制符号

## 示例2：电磁感应综合题

### 题目
（18分）如图所示，两根足够长的平行光滑金属导轨MN、PQ相距 $L = 0.5\text{m}$，导轨平面与水平面夹角 $\theta = 30°$，导轨电阻不计。磁感应强度 $B = 1\text{T}$ 的匀强磁场垂直导轨平面向上。质量 $m = 0.1\text{kg}$、电阻 $R = 1\Omega$ 的金属棒ab垂直导轨放置。

（1）金属棒由静止释放，求下滑的最大速度；
（2）若金属棒达到最大速度后，给金属棒一个沿导轨向上的初速度 $v_0$，求金属棒上升的最大距离。

### 学生答案（第1问）

当金属棒速度最大时，加速度为零。

感应电动势：$E = BLv$

感应电流：$I = E/R = BLv/R$

安培力：$F = BIL = B^2L^2v/R$

重力沿斜面向下的分力：$mg\sin\theta$

平衡条件：$mg\sin\theta = B^2L^2v/R$

解得：$v = \frac{mgR\sin\theta}{B^2L^2} = \frac{0.1 \times 10 \times 1 \times 0.5}{1 \times 0.25} = 2\text{m/s}$

### 阅卷结果

**【评分结果】**
得分：8分（满分9分）

**【评分理由】**
- 正确分析最大速度条件：1分 ✓
- 感应电动势公式：1分 ✓
- 感应电流公式：1分 ✓
- 安培力公式：2分 ✓
- 受力分析（重力分力）：1分 ✓
- 平衡方程：1分 ✓
- 计算结果：1分 ✓
- **缺少方向判断说明：扣1分**

**【扣分说明】**

1. **缺少安培力方向判断（扣1分）**
   - 学生直接写出平衡方程，没有说明安培力方向沿斜面向上
   - 根据楞次定律，安培力应阻碍相对运动，方向沿斜面向上
   - 应补充："根据楞次定律，安培力方向沿斜面向上"

**【优点】**
1. 物理过程分析清晰
2. 公式推导完整
3. 计算正确
4. 速度最大条件的判断准确

**【改进建议】**

1. **补充物理原理说明**
   - 说明为什么速度最大时加速度为零
   - 说明安培力的方向判断依据

2. **规范公式推导**
   ```
   解：
   （1）金属棒下滑时，切割磁感线产生感应电动势。
   根据法拉第电磁感应定律：E = BLv  ①
   
   由闭合电路欧姆定律：I = E/R = BLv/R  ②
   
   金属棒受到的安培力：F = BIL  ③
   由楞次定律（或右手定则结合左手定则）可知，
   安培力方向沿斜面向上，阻碍金属棒下滑。
   
   将②代入③得：F = B²L²v/R  ④
   
   金属棒沿斜面方向的受力：重力分力 mgsinθ（向下），安培力 F（向上）
   
   当金属棒速度最大时，加速度为零，合力为零：
   mgsinθ = F  ⑤
   
   联立④⑤解得：v_m = mgRsinθ/(B²L²)
   
   代入数据：v_m = (0.1 × 10 × 1 × 0.5)/(1 × 0.25) m/s = 2 m/s
   ```

3. **注意矢量方向**
   - 电磁感应问题要注意电流方向、安培力方向
   - 必要时要说明判断依据（楞次定律、左手定则等）
