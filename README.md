# 高考阅卷老师 Skill

## 简介

这是一个模拟高考阅卷老师的 AI Skill，可以按照高考评分标准批改各科试卷，支持全国高考 **9 大科目**：语文、数学、英语、物理、化学、生物、政治、历史、地理。

## 功能特点

- ✅ **全科目支持**：语数英 + 理化生 + 政史地
- ✅ **图片阅卷**：支持上传试卷照片，自动识别批改
- ✅ **标准评分**：按照官方评分细则给分
- ✅ **踩点给分**：精准识别得分点和失分点
- ✅ **步骤分计算**：理科大题按步骤给分
- ✅ **改进建议**：提供具体的答题改进建议
- ✅ **争议处理**：对创新答案合理评估

## 使用方法

### 方式一：图片上传（推荐）

直接上传试卷图片，系统自动识别并批改：

```
用户：[上传试卷照片]
用户：请帮我批改第3题

AI：
【图片识别】
题目：已知函数 f(x) = x² - 4x + 3，求函数的极值。
学生答案：f'(x) = 2x - 4 = 0，得 x = 2，f(2) = -1

【评分结果】
得分：11分（满分12分）
...
```

**图片质量建议**：
- 确保图片清晰，文字可辨认
- 尽量正面拍摄，避免倾斜和阴影
- 手写答案字迹应工整

详细说明：[docs/image_grading.md](docs/image_grading.md)

### 方式二：文字输入

直接输入题目和答案：

```
请帮我批改这道数学题：
【题目】已知函数 f(x) = x² - 2x + 3，求其在 [0,3] 上的最大值和最小值。
【学生答案】[粘贴答案]
```

### 规范输入格式

为了获得最佳评分效果，请按以下格式提供信息：

```
【科目】数学
【题型】解答题
【分值】12分
【题目】...
【学生答案】...
【标准答案】（可选）...
```

### 示例对话

**例 1：数学解答题**
```
用户：请批改这道数学题
题目：求函数 y = x³ - 3x² 的极值
学生答案：y' = 3x² - 6x，令 y' = 0 得 x = 0 或 2。极大值 f(0) = 0，极小值 f(2) = -4。

AI：
【评分结果】
得分：9分（满分12分）

【评分理由】
- 求导正确：2分 ✓
- 令导数为零：1分 ✓
- 求出临界点：1分 ✓
- 缺少二阶导数或一阶导数符号判断：扣2分
- 极值结果正确：4分 ✓
- 缺少必要的文字说明（如"令y'=0求临界点"）：扣1分

【扣分说明】
1. 判断极值类型时缺少过程：扣2分
   建议：写出 y'' = 6x - 6，或列表判断 y' 符号变化
2. 答题不够规范：扣1分
   建议：增加"解："字，关键步骤用文字说明

【改进建议】
1. 解答题要写"解"或"证明"
2. 关键步骤要配上文字说明
3. 判断极值时要有明确的方法（二阶导或符号变化）
```

**例 2：语文作文**
```
用户：请给这篇作文打分
题目：以"青春"为话题写一篇作文
学生作文：[作文内容]

AI：
【评分结果】
得分：48分（满分60分）

【等级评定】
二类卷上（48-53分区间）

【评分理由】
基础等级（38分/40分）：
- 内容（18分/20分）：中心明确，内容较充实，感情真实
- 表达（20分/20分）：语言流畅，结构完整，字迹清楚

发展等级（10分/20分）：
- 较有文采：善用比喻、排比等修辞
- 有一定的思想深度

【扣分说明】
- 错别字：发现2个，不扣分（不足3个）

【优点】
1. 立意积极向上，紧扣"青春"主题
2. 结构完整，层次清晰
3. 语言流畅，有一定文采
4. 举例恰当，有真情实感

【改进建议】
1. 可增加一两个具体事例，使内容更充实
2. 结尾可再升华，提出更有深度的思考
3. 部分语句略显平淡，可进一步增强文采
4. 注意"的、地、得"的正确使用
```

**例 3：英语作文**
```
用户：请批改这篇英语作文
题目：Write a letter to your friend Tom about your summer vacation plan.
学生作文：[作文内容]

AI：
【评分结果】
得分：22分（满分25分）

【档次评定】
第四档（22-28分：很好地完成了任务）

【评分维度】
内容要点（8分/9分）：
✓ 暑假计划描述清晰
✓ 表达期待之情
△ 可补充更多细节

词汇语法（7分/9分）：
✓ 词汇使用较丰富
✓ 语法基本准确
△ 个别时态错误
△ 存在少量中式英语

篇章结构（4分/4分）：
✓ 书信格式正确
✓ 段落清晰，过渡自然

语言得体（3分/3分）：
✓ 语气恰当，符合朋友间通信

【主要错误】
1. "I very like swimming." → "I like swimming very much."（中式英语）
2. "I will went to Beijing." → "I will go to Beijing."（时态错误）
3. "Please write me soon." → "Please write to me soon."（介词遗漏）

【改进建议】
1. 注意时态一致性
2. 避免中式英语表达
3. 可增加更多细节描写
4. 丰富句型结构，适当使用复合句
```

## 评分原则

### 1. 踩点给分原则
- 答案中有得分点关键词即可得分
- 不要求与标准答案完全一致
- 意思对即可，专业术语必须准确

### 2. 步骤分原则（理科）
- 写出关键公式/方程：得基础分
- 有正确的推导过程：得过程分
- 得出正确结果：得结果分
- 过程正确但结果错：扣结果分

### 3. 多答不扣分原则
- 除非题目特别说明，多答不扣分
- 但多答内容错误可能影响评价

### 4. 公平公正原则
- 严格按照评分标准
- 一把尺子量到底
- 不冤枉任何一份试卷

## 各科目评分特点

| 科目 | 主要题型 | 评分特点 |
|------|----------|----------|
| 语文 | 阅读、作文 | 踩点给分，作文分档评分 |
| 数学 | 选填、解答 | 步骤分严格，按步骤给分 |
| 英语 | 选择、作文 | 语法填空注意词形变化 |
| 物理 | 选择、计算 | 注意单位和有效数字 |
| 化学 | 选择、实验 | 方程式配平和条件 |
| 生物 | 选择、简答 | 专业术语必须准确 |
| 政治 | 选择、论述 | 观点正确，联系材料 |
| 历史 | 选择、材料 | 史实准确，史论结合 |
| 地理 | 选择、综合 | 自然重原理，人文重因素 |

## 注意事项

1. **输入完整题目**：提供完整的题目内容有助于准确评分
2. **注明分值**：如果有明确分值，请告知，便于按比例评分
3. **提供标准答案**（可选）：如有标准答案，可提供更精准的对比评分
4. **创新答案**：如果你的答案与标准答案不同但合理，会进行评估
