# 图片阅卷使用指南

## 功能简介

高考阅卷老师 Skill 支持直接上传试卷图片进行阅卷。系统会自动识别图片中的文字内容，并按照高考评分标准进行专业批改。

---

## 使用方式

### 1. 直接发送图片

在对话中直接上传试卷图片（支持照片或扫描件），系统会自动处理并给出评分。

```
用户：[上传图片]
用户：请帮我批改这道题

系统：收到图片，正在识别内容...

【识别结果】
题目：已知函数 f(x) = x² - 4x + 3，求函数的极值。
答案：f'(x) = 2x - 4 = 0，得 x = 2，f(2) = -1

【评分结果】
得分：11分（满分12分）
...
```

### 2. 指定题目范围

如果图片中包含多道题目，可以指定要批改的题号：

```
用户：[上传图片]
用户：请批改第3题和第5题
```

### 3. 分批上传

对于多页试卷，可以分批发送图片：

```
用户：[上传图片1]
用户：这是第1页

用户：[上传图片2]
用户：这是第2页，请一起批改
```

## 图片质量要求

### 推荐标准

| 要求 | 说明 | 影响 |
|------|------|------|
| **清晰度** | 文字清晰可辨认，无模糊 | 识别准确率的关键 |
| **光线** | 光线充足均匀，无阴影 | 避免识别错误 |
| **角度** | 正面拍摄，无明显倾斜 | 确保文字方向正确 |
| **背景** | 背景简单，与文字对比明显 | 提高识别精度 |
| **分辨率** | 建议 1080p 以上 | 细节更清晰 |

### 拍摄建议

**正确示例**：
- 手机垂直于试卷，距离 20-30cm
- 确保试卷平整无褶皱
- 在自然光或台灯下拍摄

**常见问题**：
- ❌ 图片过暗或过亮
- ❌ 角度倾斜导致变形
- ❌ 手影遮挡文字
- ❌ 图片模糊或有噪点

## 不同科目图片处理

### 数学

**特点**：
- 包含大量公式和符号
- 手写答案需要清晰书写

**建议**：
- 公式部分尽量书写规范
- 分数、根号、指数等符号写清楚
- 可以使用文字补充复杂公式

**示例**：
```
用户：[上传数学答题图片]

系统：识别结果如下：
题目：求导 f(x) = x³ - 3x²
学生答案：f'(x) = 3x² - 6x

评分：答案正确，得满分。
```

### 语文

**特点**：
- 文字量大，作文篇幅长
- 书写工整度影响识别

**建议**：
- 字迹工整，避免连笔
- 段落清晰，标点规范
- 作文建议分段拍摄

**示例**：
```
用户：[上传作文图片]

系统：识别出作文内容（约800字）
【评分结果】
得分：45分（满分60分）
等级：二类卷
...
```

### 英语

**特点**：
- 英文字母识别相对准确
- 注意大小写区分

**建议**：
- 字母书写清晰，避免混淆（如 a/u, n/u）
- 标点符号使用英文标点

### 理科（物理/化学/生物）

**特点**：
- 包含公式、符号、单位
- 可能有图表

**建议**：
- 单位符号写清楚（如 Ω、℃、mol）
- 化学方程式配平要清晰
- 图表可以用文字补充说明

### 文科（政治/历史/地理）

**特点**：
- 文字量大，论述性强
- 需要条理清晰

**建议**：
- 分点作答，使用序号
- 关键术语书写规范
- 条理清晰，层次分明

## 识别局限性

### 可能识别困难的情况

1. **字迹潦草**
   - 连笔字难以识别
   - 草书、行书识别率低
   - 建议：尽量写楷书或行楷

2. **复杂公式**
   - 多层嵌套的公式
   - 特殊数学符号
   - 建议：用文字说明公式含义

3. **图表图形**
   - 几何图形
   - 函数图像
   - 建议：用文字描述图形特征

4. **涂改过多**
   - 大面积涂改
   - 修改痕迹覆盖原字
   - 建议：保持卷面整洁

### 识别错误处理

如果识别结果有误：

```
用户：[上传图片]

系统：识别内容如下：
...

用户：第2行识别错了，应该是 x² 不是 x3

系统：已更正，重新评分...
```

## 最佳实践

### 单题拍摄

一道题一张图片，效果最佳：
- 识别准确率高
- 评分更精准
- 便于针对性反馈

### 文字补充

对于识别困难的内容，用文字补充：

```
用户：[上传图片]
用户：图片中的公式是 ∫(x²+1)dx
```

### 分步批改

对于多步骤的题目，可以要求分步评分：

```
用户：[上传图片]
用户：请分步骤批改这道数学题
```

## 常见问题

### Q1: 图片识别准确率有多高？

A: 印刷体识别率 > 95%，工整手写 > 85%，潦草手写 > 60%。建议：
- 印刷试卷：直接使用
- 工整手写：放心使用
- 潦草手写：建议配合文字说明

### Q2: 支持哪些图片格式？

A: 常见格式均支持：
- JPG / JPEG
- PNG
- GIF
- WebP
- HEIC（部分平台）

### Q3: 图片大小有限制吗？

A: 建议单张图片不超过 10MB，分辨率不超过 4K。过大的图片可能被压缩。

### Q4: 手写作文能识别吗？

A: 可以，但需要：
- 字迹相对工整
- 避免过度连笔
- 标点符号清晰

### Q5: 识别错误怎么办？

A: 可以通过以下方式纠正：
1. 指出具体错误位置
2. 提供正确的文字内容
3. 重新进行评分

### Q6: 图片中的答题卡能识别吗？

A: 可以识别，但：
- 选择题可能需要人工确认选项
- 填空题注意答题位置
- 建议配合文字说明

## 优化建议

### 对学生

1. **平时练习**
   - 养成工整书写习惯
   - 规范使用标点符号
   - 分点作答，条理清晰

2. **拍照上传**
   - 检查图片清晰度
   - 确认内容完整
   - 及时纠正识别错误

3. **结合使用**
   - 复杂公式用文字补充
   - 图表用文字描述
   - 不确定时多沟通

### 对家长

1. **协助拍摄**
   - 提供良好的光线环境
   - 帮助调整拍摄角度
   - 检查图片质量

2. **结果核对**
   - 核对识别结果
   - 发现错误及时指出
   - 关注评分反馈

## 技术说明

### OCR 识别流程

```
图片上传
    ↓
图像预处理（去噪、增强、矫正）
    ↓
文字区域检测
    ↓
文字识别（OCR）
    ↓
公式识别（LaTeX）
    ↓
结果整合与优化
    ↓
输出识别文本
    ↓
人工确认（如有必要）
    ↓
进入阅卷流程
```

## 使用示例

### 示例 1：数学解答题

**用户上传**：数学题照片

**系统处理**：
```
【图片识别】
题目：求函数 y = x³ - 3x² 的极值
学生答案：
y' = 3x² - 6x
令 y' = 0，得 x = 0 或 2
f(0) = 0，f(2) = -4
极大值是0，极小值是-4

【评分结果】
得分：9分（满分12分）

【评分理由】
✓ 求导正确：2分
✓ 求临界点：2分
✓ 计算函数值：2分
✗ 极值判断过程缺失：扣3分
✗ 答题不够规范：扣1分

【改进建议】
1. 补充极值判断过程（二阶导数或符号法）
2. 规范答题格式，增加文字说明
```

### 示例 2：语文作文

**用户上传**：手写作文照片

**系统处理**：
```
【图片识别】
识别字数：约 820 字

【评分结果】
得分：45分（满分60分）
等级：二类卷

【识别备注】
发现2处字迹难以辨认（已标注）

【评分理由】
内容：16/20 分
表达：17/20 分
发展：12/20 分

【改进建议】
1. 字迹可以更工整，避免连笔
2. 增加论据，丰富内容
3. 结尾升华主题
```

## 注意事项

1. **隐私保护**
   - 图片内容仅用于阅卷评分
   - 不会存储或分享原始图片
   - 识别结果仅当前对话有效

2. **识别限制**
   - 极度潦草的字迹可能无法识别
   - 复杂图形需要文字辅助
   - 特殊符号可能识别错误

3. **结果参考**
   - 图片阅卷结果仅供参考
   - 最终以人工核对为准
   - 建议重要考试使用文字输入
